## System Requirements

```DeepGFT``` can run on Linux and Windows. The package has been tested on the following systems:

- Linux: CentOS 7
- Windows: Windows 10

```DeepGFT``` requires python version >= 3.7. We tested it in python 3.8 and cuda 11.6.1 on Linux.
